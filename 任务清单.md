## 阶段 I：OpenCode 集成（搁置 · 与子 Agent 合并规划）

> **状态：** 📋 已排期，**暂不开发**。OpenCode 为 **MIT 开源** AI 编码 Agent（[opencode.ai](https://opencode.ai/) · [anomalyco/opencode](https://github.com/anomalyco/opencode)），与自研 Sheldon **可以并存**，适合作为「专业编码子 Agent」，而非替换主 Loop。  
> **集成面：** `opencode serve`（HTTP + OpenAPI）· `opencode run`（CLI）· `@opencode-ai/sdk` · ACP（stdin JSON-RPC）。

### 为什么可行？

| 方式 | 说明 |
|------|------|
| **子 Agent 工具** | Sheldon 主 Agent 调用 `opencode_delegate(task)`，OpenCode 在指定 `--dir` 执行，结果回写对话 |
| **HTTP  sidecar** | 容器/本机常开 `opencode serve --port 4096`，Sheldon 用 REST/SDK 发 session/message |
| **CLI 一次性** | `opencode run --attach http://127.0.0.1:4096 "..."` 适合 CI/脚本 |

OpenCode 自带 Build/Plan/Explore 等 **subagent**，与任务清单里的 `delegate.py` 方向一致：**Sheldon 编排 + OpenCode 干重活**。

### 主要风险（规划时记牢）

- **双运行时**：OpenCode 偏 Node/Bun；Sheldon 是 Python — 部署要装两套或 Docker 多阶段
- **双沙箱**：工作目录、权限确认需统一（Sheldon 三档权限 vs OpenCode permission）
- **双模型账单**：OpenCode 可接百炼/OpenAI 兼容端点，但要单独配 provider，避免和 Qwen 路由打架
- **Windows**：OpenCode Windows 支持仍在完善，ECS/Linux 部署更稳

## 阶段 L：用户文件存储 · 配额（规划）

> **用户问「文件存在哪」：** 已在 **`runtime/workspaces/{user_id}/projects/`**（沙箱）；会话在 SQLite + `sessions/` 目录。

### 存储布局（生产 ECS 建议）

```text
/opt/sheldon-agent/runtime/          # Docker volume 挂载
├── db/sessions.sqlite               # 全员会话（按 user_id 列隔离）
├── workspaces/{user_id}/projects/   # 用户 Agent 读写的文件
├── workspaces/{user_id}/sessions/   # CLI 本地 JSON 持久化（可选）
```

## 阶段 N：桌面软件 · Cursor 式 EXE（规划 · 产品方向）

> **目标：** 做成 **双击即用的 Windows 桌面软件**（`.exe` 安装包），UI 接近 Cursor：对话中可见思考与工具/MCP 调用、左侧文件树、内置浏览器预览。  
> **技术选型：** **Electron**（主进程 + WebView + 内置浏览器）+ **PyInstaller** 打包 Python 后端 + 复用 `apps/web`（改造布局）。  
> **与公网 Web 关系：** 共用 `core/` · `server/` · SSE 事件；桌面版默认 **单机本地模式**（可跳过登录），数据在 `%AppData%/SheldonAgent/runtime/`。  
> **参考讨论：** 2026-07-05 — 用户希望 Agent 写代码可见、对话内展示工具链、像 Cursor 打开项目。

### 架构一览

```text
Sheldon Agent.exe
├── Electron 主进程（窗口 · 启停后端 · 内置 BrowserView）
├── 界面 apps/web（Cursor 三栏/四栏布局）
├── sheldon-server.exe（PyInstaller 打包 FastAPI + Agent）
└── 数据 %AppData%/SheldonAgent/runtime/
    ├── workspaces/{user_id}/projects/   ← Agent 读写代码
    ├── db/sessions.sqlite
    └── logs/{date}/{user_id}.jsonl
```

### 目标界面（Cursor 式）

```text
┌──────────────┬────────────────────────────┬─────────────────┐
│  左栏         │  中间：对话                 │  右栏（可折叠）   │
│  · 会话列表   │  · 用户 / Agent 消息        │  · MCP 连接状态 │
│  · 项目文件树 │  · 内嵌「思考步骤」卡片      │  · 活动/工具流   │
│  · 打开文件夹 │  · 内嵌「工具/MCP 调用」卡片 │                 │
├──────────────┴────────────────────────────┴─────────────────┤
│  底栏 Tab：终端输出 | 内置浏览器（预览本地 HTML / localhost）   │
```
