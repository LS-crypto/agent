# Sheldon Agent 使用说明

> 在线地址：**http://118.178.144.109/**  
> 适用：通过浏览器使用 Sheldon 编程 Agent 的用户

---

## 一、这是什么？

**Sheldon Agent** 是基于阿里云百炼 Qwen 大模型的编程助手，可以：

- 回答编程与技术问题  
- 读写沙箱内文件、执行命令（需确认时会有弹窗）  
- 查询 GitHub 仓库 Issue / PR（管理员已配置时）  
- 分步展示 Agent 思考与工具调用过程  

请使用 **Chrome、Edge、Firefox、Safari** 等现代浏览器访问，无需安装客户端。

---

## 二、如何打开？

1. 在浏览器地址栏输入：  
   **http://118.178.144.109/**
2. 看到 **「有什么可以帮你？」** 欢迎页即表示进入成功。  
3. 若打不开，请检查网络；仍失败请联系管理员。

## 三、界面说明

```text
┌─────────────┬──────────────────────────┬─────────────┐
│  左侧栏     │       中间：对话区        │  右侧栏     │
│  会话列表   │  消息 + 输入框            │  Agent 活动 │
│  新建对话   │  模型 / 权限 / 发送       │  工具时间线 │
└─────────────┴──────────────────────────┴─────────────┘
```

| 区域 | 功能 |
|------|------|
| **左侧栏** | 历史会话；点击切换；「新建对话」开始新话题 |
| **中间** | 与 Agent 对话；支持 Markdown（标题、列表、引用、表格、代码块） |
| **右侧 Agent 面板** | 运行中显示工具调用、分步推理（Max 模型）等活动 |
| **顶栏** | 主题切换（深色 / 浅色 / 跟随系统）、清空当前会话 |

**小屏 / 手机：** 侧栏与 Agent 面板可通过菜单按钮展开，以 overlay 形式显示。

## 四、基本操作

### 发送消息

- **Enter** — 发送  
- **Shift + Enter** — 换行（不发送）  

### 快捷键

| 快捷键 | 作用 |
|--------|------|
| **Ctrl + L** | 聚焦到输入框 |
| **Ctrl + K** | 清空当前会话所有消息 |

（Mac 用户可将 Ctrl 换为 ⌘，视浏览器而定。）

### 新建与清空

- 左侧 **新建对话** — 开始新会话，旧会话仍保留在列表中  
- 顶栏 **清空对话** — 仅清空**当前会话**的消息，不可恢复  

### 主题

顶栏可切换：

- **深色** — 默认 Cursor 风格  
- **浅色** — 明亮模式  
- **跟随系统** — 随操作系统深浅色自动切换

## 五、模型与权限

输入框左下角有两个下拉选项：

### 模型

- **自动路由** — 系统按问题复杂度选择合适模型（推荐）  
- 也可固定选择具体 Qwen 模型（如 Plus / Max 等，以界面列表为准）  

使用 **Max** 类模型时，右侧 Agent 面板可能出现 **分步思考** 时间线。

### 权限档位

| 档位 | 说明 |
|------|------|
| **保守** | 只读类操作自动执行；写入、命令、技能加载等需你确认 |
| **平衡** | 默认；读操作自动；写文件、执行命令、Git 提交需确认 |
| **宽松** | 读与系统信息、技能查询自动；写入与命令仍需确认 |

当 Agent 要执行敏感操作（如写文件、跑 Shell）时，会弹出 **确认对话框**，请阅读说明后选择 **允许** 或 **拒绝**。

## 六、可以问什么？（示例）

欢迎页提供快捷入口，也可直接输入，例如：

- 「列出当前项目目录结构」  
- 「在沙箱里创建一个 hello.py」  
- 「查看 MCP 与工具连接状态」  
- 「在 GitHub 仓库 LS-crypto/agent 里搜 open 的 Issue」  
- 「解释一下这段代码……」（粘贴代码）  

Agent 在**服务端沙箱**中操作文件，不会直接访问你本机电脑。

## 七、常见问题

### 1. 页面一直显示「后端未就绪」

- 服务可能正在重启，等待 1～2 分钟后刷新。  
- 仍不行请联系管理员检查 Docker / Nginx 状态。

### 2. 发送后长时间无回复

- 复杂任务或调用工具时可能需等待数十秒。  
- 顶栏出现 **「Agent 运行中」** 或 **「等待你的确认…」** 属正常。  
- 若超过 5 分钟无响应，刷新页面并重试。

### 3. 弹出确认框是什么意思？

Agent 即将执行可能有影响的操作（写文件、运行命令等）。请看清 **操作说明** 再决定允许或拒绝。

### 4. 会话会保存吗？

会。同一浏览器下会话列表会保留在服务器上（当前版本为共享环境，**尚未区分个人账号**）。

### 5. 支持手机吗？

支持基本访问；复杂操作建议使用桌面浏览器，体验更好。

## 八、使用须知

1. **请勿输入密码、私钥、银行卡等敏感信息。**  
2. **AI 回答仅供参考**，重要代码与命令请自行审查后再用于生产环境。  
3. 当前版本 **无需登录**，链接知道即可访问；请勿随意传播链接（见管理员说明）。  
4. 滥用或恶意刷接口可能导致服务被管理员限制访问。
